# @orchestra-mcp/ai

AI chat and copilot components for Orchestra MCP

Part of the [Orchestra MCP](https://orchestra-mcp.dev) ecosystem.

## Install

```bash
npm install @orchestra-mcp/ai
# or
pnpm add @orchestra-mcp/ai
```

## Usage

```tsx
import { } from '@orchestra-mcp/ai'
```

## Requirements

- React >= 18.0.0
